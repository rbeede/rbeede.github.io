Issue

After resuming from sleep the wired Ethernet network adapter was either missing or always reported media disconnected.
Driver Attempt
Latest driver (Realtek DNIS or NetAdapterCx, both "Not Support Power Saving" or other) did not help with the issue. The driver from the OEM motherboard website also had the same issue.

Solution

  1. Device Manager
  2. Network adapters
  3. If needed, Scan for hardware changes
  4. Right-click for Properties on the network adapter
  5. Advanced tab
  6. Advanced EEE = Disabled
  7. Energy-Efficient Ethernet = Disabled
  8. Green Ethernet = Disabled
    • This was the one I had to change specifically
  9. Power Saving Mode = Disabled
Putting the computer to sleep and then waking it up the network adapter worked normally.
