--- creation-epoch: 1325544964 lastmod-epoch: 1325544967 categories: ['Tech Tricks'] title: Custom icon for USB device ---
  1. Pick an image and cut out a part so it is no wider or taller than 64x64 pixels. Save as PNG.
  2. Go to http://www.convertico.com/ and convert the PNG to ICO
  3. Copy the ICO file to the root of the usb drive and name it diskicon.ico
  4. Open a text editor and enter the following:
    [autorun]
icon=diskicon.ico
  5. Save the file in the root of the usb drive as autorun.inf
  6. Make diskicon.ico and autorun.inf files hidden if you like
  7. Safely eject the usb
  8. Re-insert
  9. Open Computer (My Computer) and admire
