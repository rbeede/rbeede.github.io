--- creation_epoch: 1673032251 lastmod_epoch: 1673032251 categories: ['Computer Problems'] title: VMware VM Freezes on-Workstation Linux Host.html --- Had an Ubuntu 20 VM running in VMware Workstation Pro version 16.2.5 that would freeze every few hours for about 20 seconds. Tried rebuilding the VM without much success. This freezing would occur even if the host had no disk usage, cpu usage, and plenty of memory. Similar for the guest VM with low cpu usage tasks and memory. Simple things such as opening a new web browser tab in the guest VM would just randomly freeze up.

I tried the last solution at VMWare Guest periodic freezes..then comes back post by adamitj on Dec 7, 2022.

Power down the VM and add to the vmx: 

isolation.tools.unity.disable = "TRUE"

Since this was a Linux desktop guest VM and a Linux host hypervisor the Unity feature wasn't available anyway. Disabling this setting appears to have cleared up the freezing.
