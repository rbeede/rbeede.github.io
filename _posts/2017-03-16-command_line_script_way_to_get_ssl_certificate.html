--- creation-epoch: 1489681261 lastmod-epoch: 1534942592 categories: ['Tech Tricks'] title: Command line script way to get SSL certificate --- 
openssl s_client -prexit -showcerts -connect www.rodneybeede.com:443 2>/dev/null </dev/null| sed -ne '/BEGIN CERT/,/END CERT/p'

The 2>/dev/null removes output from the openssl command so you can get only the ASCII BASE64 encoded certificate.

The </dev/null disables STDIN input so the openssl command terminates after a successful connection.
