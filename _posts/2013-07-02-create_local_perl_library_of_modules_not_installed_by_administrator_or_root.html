--- creation_epoch: 1372778889 lastmod_epoch: 1372778889 categories: Programming title: Create local Perl library of modules not installed by administrator or root --- Local::Lib is the Perl module you want. Combine that with Dynamically load Perl library modules based on script location to get local-lib into your Perl script, and you should find it easier to download a missing module along with all of its dependencies.

Note that for modules that have compiled code you are still looking at platform specific builds.
